# Playwright TypeScript Automation Framework

## Overview
End-to-end UI automation framework built using Playwright with TypeScript. The framework supports parallel execution and modern web application testing.

## Features
- Page Object Model
- Parallel Test Execution
- HTML Reporting
- Cross Browser Testing
- Headless execution support

## Tech Stack
Playwright  
TypeScript  
Node.js  

## Framework Structure
tests
pages
utils
fixtures

## Sample Test Scenarios
- Login functionality
- Form submission validation
- Dashboard navigation
- User workflow automation

## How to Run Tests
Install dependencies:
npm install

Run tests:

npx playwright test# QA_Vishal
